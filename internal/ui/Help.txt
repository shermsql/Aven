Aven Provides Caesar And XOR Encryption Through A Clean, Modular CLI — Built For Speed And Extensibility.

Usage:
	Aven -m <Mode> -a <Algorithm> -t <Text>

Parameters:
	-m Mode Of Operation    Encrypt / Decrypt
	-a Algorithm To Use     Caesar / XOR
	-t Text To Process      "Hello World"
	-h Show Help Message    Displays This Output

Examples:
	Aven -m Encrypt -a Caesar -t "Hello World!"
	Aven -m Decrypt -a Caesar -t "Khoor Zruog!"
	Aven -m Encrypt -a XOR -t "Secret 123"
	Aven -m Decrypt -a XOR -t "<Encrypted Output>"
	Aven -h

Tip:
	Caesar Uses A Default Shift Of "3"
	XOR Uses A Default Key Of "7"
